# Synthetic Glyph demonstration

This fixture is not a real paper or investment analysis. `source.ts` is the
versioned source of its PDF text, evidence coordinates, claims, report,
concepts, visuals, and distribution previews. Run `pnpm generate:fixture` from
the repository root to regenerate `paper.pdf`, `edition.json`, and the public
demo PDF. Evidence coordinates are calculated from the same drawing operations
that generate the PDF; they are never estimated by eye.
